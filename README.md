### About RAGNAROK ###

 Ragnarok is a library for solving lattice boltzmann method. The code is open-source and developed by Lento Manickathan.

### Installation ###

#### via pip
```shell
pip install -e ragnarok
```

### Tutorials ###

#### 1D : shock-tube flow
```shell
python shock_tube.py
```

#### 2D: lid-driven cavity flow
```shell
python cavity.py
```
